# 📄 Offer Generator

A web application for creating and exporting client offers/quotes as PDF documents.

---

## 🚀 Features

- Add products to an offer with custom prices and units
- Edit product values inline
- Export the offer as a formatted PDF
- Duplicate product prevention
- Responsive design for mobile and desktop

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Framework | [Next.js](https://nextjs.org/) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Select | [react-select](https://react-select.com/home) |
| PDF Generation | [@react-pdf/renderer](https://react-pdf.org/) |
| UI Components | shadcn/ui |

---

## 📦 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/your-username/offer-generator.git
cd offer-generator
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```
